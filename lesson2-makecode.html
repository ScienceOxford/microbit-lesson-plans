Lesson 1: micro:bit thermometer


Step 1: Recap from Lesson 1

It is worth spending a few minutes reminding students how to display images on the screen, and how to download and run their programs on the micro:bit.


Step 2: Displaying the Temperature

The current temperature is not one set value, it changes. In computer programming, we call a changeable value a variable.

BBC Bitesize - How do computer programs use variables?

Run most of this lesson using the Simulator, so that your students can easily change the 'current temperature' to test their code.


To display the temperature, students still only need to use the blocks in the Basic and Input menus, but they will be combined together in new ways.

The show number 0 block in the Basic menu scrolls numbers across the micro:bit's display - your students may have used this block during lesson 1. Place this block into a forever loop.

As well as changing the number in this block, an Input can be placed within it, to display this instead.

Find the temperature (°C) block, and slot this into the number space within the show number 0 block.

Extension ideas

  • What happens if you place the new code inside on start instead of forever? Change the 'current temperature' using the Simulator to test the code.
  • Use some of the other blocks used in Lesson 1, such as pause and on button A pressed to make a more interesting program.
  • Try the other, similar, blocks within the Input menu, such as light level and compass heading (°).

Step 3: Selection

Now you will introduce the concept of Selection in computer programming.

BBC Bitesize - How do we get computers to do what we want?

These are if this then that statements, and they are found in the Logic menu.

Combine an if true then... else block with a 0 = 0 block and your temperature input, to show one thing if the temperature is equal to a given number, and otherwise something else.

Extension idea - more complex statements

Try using the other blocks in the Logic menu to make more complex statements e.g.

  • if temperature > 20
  • if not temperature = 25
  • if temperature = 25 and button A is pressed...

You can add extra conditions to the statement by using the + symbol to add an else if ... then option.


Extension idea - calibrating the temperature

Set up two thermometers in different locations (e.g. by a radiator and the window).

Students can take their micro:bit over to that location and compare the temperature on the thermometer to the temperature shown by the micro:bit (note, it may take some time for the micro:bit to adjust to the new temperature).

By working out the difference, they can use the blocks in the Math menu to calibrate their micro:bit to show a more accurate temperature reading.
